# Shop

Simple Ruby imitation store program

# Custom

You can add your own porducts in `/data/*` folder. Txt file format:

~~~
1984 // title
Антиутопическая литература // genre
Дж. Оруэлл // author
1001 // price
10 // amount
~~~

# How to run

Clone repo:

~~~
git clone https://github.com/orekorekorek/shop.git
~~~

Enjoy:

~~~
ruby main.rb
~~~
